📘 Análisis y Diseño de Algoritmos

Repositorio del curso Análisis y Diseño de Algoritmos, enfocado en el estudio teórico y práctico de técnicas algorítmicas, análisis de complejidad y resolución eficiente de problemas computacionales.

🎯 Objetivos del curso

Diseñar algoritmos correctos y eficientes para distintos tipos de problemas.

Analizar el tiempo y espacio de ejecución usando notación asintótica.

Comparar distintas estrategias algorítmicas.

Implementar soluciones óptimas en diferentes lenguajes de programación.
